Dela verktyget

För att dela en riktig länk: döp filen till index.html och publicera den på GitHub Pages, Netlify eller Cloudflare Pages.

  1. Skapa ett nytt GitHub-repository, t.ex. italiensk-tranare.
  2. Ladda upp filen som index.html.
  3. Gå till Settings → Pages.
  4. Välj branch main och folder /root.
  5. Dela länken GitHub visar.
Det är normalt okej att dela ett eget studieverktyg. Tänk bara på att du bör ha rätt att publicera själva ordlistorna/verbdata om länken är publik. Varje användares statistik sparas lokalt i deras egen webbläsare.
